# Twittor 
Twittor is a simple social media clon of Twitter. \
\
<a href="https://twittor-frontend.herokuapp.com/login">Click here to go to Twittor</a>

## Technologies used

* React 
* Nodejs
* MongoDB

## Running 

### Backed 
At twittor/backend
```
npm run start
```

### Frontend 
At twittor/frontend
```
npm run start
```
